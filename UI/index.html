ECE Dept Christmas Tree

This is a way for everyone to control the pattern of the ECE department Christmas Tree.

You can either choose a preselected pattern or write some code using the given functions to change the lights.

Once you have made your decision please click the "Add to Queue" button and you pattern will be played when it is it's turn.
